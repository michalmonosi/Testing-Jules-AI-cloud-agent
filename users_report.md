# User Data Analysis Report

This report provides a basic analysis of the user data from the `users_sql.csv`
file. The dataset includes user information such as name, occupation, and
salary.

## Salary Statistics

The following statistics were calculated for the user salaries:

- **Mean Salary:** $97,830.00
- **Median Salary:** $90,000.00
- **Standard Deviation of Salary:** $41,519.15

## Observations

The data indicates a wide range of salaries, with a mean salary of
approximately $97,830. The median salary is slightly lower at $90,000,
which suggests that there are some higher salaries pulling the mean up. The
standard deviation of over $41,000 confirms the significant spread in salary
values.
